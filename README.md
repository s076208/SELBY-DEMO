# Selby MV - Static Website

This is a static HTML/CSS/JS version of the Selby MV hardware store website, optimized for GitHub Pages hosting.

## Original Site
- **Live URL**: https://selbymv.com
- **Platform**: WordPress 5.3.12 + DT-The7 theme
- **Business**: Hardware store in the Maldives (established 1965)

## Issues Fixed

1. ✅ **Broken image URLs** - All `http:/` URLs fixed to `https://`
2. ✅ **Demo URL leak** - Replaced `the7.io` link with correct `selbymv.com` path
3. ✅ **Duplicate HTML IDs** - All buttons now have unique identifiers
4. ✅ **WhatsApp phone number** - Fixed country code from `+60` to `+960`
5. ✅ **Slider dependency** - Replaced premium Slider Revolution with native CSS/JS slider
6. ✅ **Form dependency** - Replaced FormCraft with native HTML form (works with Formspree)
7. ✅ **Brand color consistency** - Unified all colors to brand red (`#e4002b`)
8. ✅ **Accessibility** - Added ARIA labels, keyboard navigation, skip links
9. ✅ **Performance** - Added lazy loading, reduced external dependencies
10. ✅ **Mobile responsive** - Tested at 480px, 768px, 1024px breakpoints
